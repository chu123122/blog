# 博客重设计 SPEC

> 技术栈：Astro
> 风格：日式/二次元
> 部署：GitHub Pages
> 分支：redesign-astro

---

## 现状

- Hexo 7.3.0 + 自定义 Vue 主题
- 15 篇文章（2024.01 - 2025.10）
- 分类：游戏开发学习记录、CS公开课学习记录、GAMES101学习记录、技术记录
- 标签：DOTS、计算机学习、设计模式、网站开发
- 素材：头像(二次元角色)、多张二次元封面图(鲸落地/心灯探梦/守护甜心/崩坠/无衔之尾)
- CDN：zstatic.net + fonts.googleapis.cn

## 目标

1. 从 Hexo 迁移到 Astro
2. 日式/二次元视觉风格
3. 保留所有 15 篇文章
4. GitHub Pages 部署
5. 代码高亮（Shiki 内置）
6. 响应式设计
7. 深色主题为主

## 设计方向

基于现有素材（鲸落地.png 等），主打：
- **深色基调** + 星空/海洋渐变色
- **毛玻璃效果** 卡片
- **柔和的强调色**（参考鲸落地配色：深蓝 #0a1628 → 蓝紫 #1a2744 → 暖橙 #e8956a）
- 二次元头像展示
- 文章卡片带封面图

## 页面结构

1. **首页** - Hero 区域（头像+介绍）+ 最新文章列表
2. **文章列表页** - 分类筛选 + 卡片网格
3. **文章详情页** - Markdown 渲染 + 目录
4. **关于页** - 个人介绍

## 实现步骤

1. Astro 项目初始化 + 基础配置
2. 全局样式 + 布局组件
3. 首页 + 文章列表
4. 文章详情页 + Markdown 渲染
5. 15 篇旧文章迁移
6. GitHub Pages 部署配置
