GAMES101学习之路(二)：光栅化的原理

2025/7/14

目的

透视除法**(Perspective Divide)**

通过投影变换的物体会经过一定的畸变，通过用坐标的w分量去除x，y，z，**归一化设备坐标(Normalized Device Coordinates, NDC)**。这是一个 -1 到 1 的立方体空间。

视口变换**(Viewport Transform)**

物体在经过透视除法后已经成功获得了一个标准[-1，1]三次方空间内的三角形，为了将这个标准空间转化为对应的不同分辨率屏幕上，我们需要经过视口变换，将经过MVP变换的[-1,1]三次方转化为WxH分辨率的变换。（平移矩阵x缩放矩阵）

光栅化器

在这之后计算机通过光栅化器，对三角形内部的点进行采样，通过重心关系和三顶点计算出纹理空间(插值、视图空间插值、法线插值等。并存储在内部供后面片元着色器使用，从而将计算出颜色。

void rst::rasterizer::rasterize_triangle(const Triangle &t, const std::array<Eigen::Vector3f, 3> &view_pos)
{
    // 对于三角形范围内的所有点
    auto v = t.toVector4();

    float min_x = INT_MAX;
    float min_y = INT_MAX;

    float max_x = INT_MIN;
    float max_y = INT_MIN;

    for (Vector4f i : v)
    {
        max_x = MAX(max_x, i.x());
        max_y = MAX(max_y, i.y());

        min_x = MIN(min_x, i.x());
        min_y = MIN(min_y, i.y());
    }
    for (int x = min_x; x <= max_x; x++)
    {
        for (int y = min_y; y <= max_y; y++)
        {
            if (insideTriangle(x, y, t.v)) // 在三角形内部
            {
                auto [alpha, beta, gamma] = computeBarycentric2D(x, y, t.v);
                float Z = 1.0 / (alpha / v[0].w() + beta / v[1].w() + gamma / v[2].w());
                float zp = alpha * v[0].z() / v[0].w() + beta * v[1].z() / v[1].w() + gamma * v[2].z() / v[2].w();
                zp *= Z;

                float origin_depth = depth_buf[get_index(x, y)];
                if (zp < origin_depth)
                {
                    depth_buf[get_index(x, y)] = zp;

                    // 颜色插值
                    Vector3f interpolated_color = interpolate(alpha, beta, gamma, t.color[0], t.color[1], t.color[2], Z);
                    // 法线插值
                    Vector3f interpolated_normal = interpolate(alpha, beta, gamma, t.normal[0], t.normal[1], t.normal[2], Z);
                    // 片元在纹理上的坐标插值
                    Vector2f interpolated_texcoords = interpolate(alpha, beta, gamma, t.tex_coords[0], t.tex_coords[1], t.tex_coords[2], Z);
                    // 片元在视图空间中的坐标查插值
                    Vector3f interpolated_shadingcoords = interpolate(alpha, beta, gamma, view_pos[0], view_pos[1], view_pos[2], Z);

                    fragment_shader_payload payload(interpolated_color, interpolated_normal.normalized(), interpolated_texcoords, texture ? &*texture : nullptr);
                    payload.view_pos = interpolated_shadingcoords;
                    Vector3f pixel_color = fragment_shader(payload);
                    set_pixel(Vector2i(x, y), pixel_color);
                }
            }
        }
    }
}
